# simple-action-game-2Dversion


lastest update : 2022-10-21

### 目前已实现功能：
- 角色操作
  - 基础移动和轻重攻击
  - airdash / 二段跳
  - 瞬闪
  - 钩锁（将自己拉向或拉回target)
  - 浮空连击（已添加渐变时间限制）
    - 浮空状态：将敌人向下砸落
    - 浮空状态：将敌人向某可选角度击飞
- 打击感
  - 镜头震动与时停（部分攻击已添加双段时停）
  - 敌人受击特效
